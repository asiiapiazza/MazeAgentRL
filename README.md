## Note per l'utilizzo
- La scena principale è `Main`, da cui è possibile testare il proprio modello allenato
  - Per assegnare all'agente il proprio modello, è necessario sostituirlo nel campo `Model` nel prefab `BearFinale`
    ![image](https://github.com/user-attachments/assets/5f373402-b4f4-4093-bbc1-64f7b9ee785c)
- E' possibile allenare il proprio modello una volta installate le necessarie librerie. Per un tutorial completo: https://unity-technologies.github.io/ml-agents/Installation/
 - Sono disponili 3 scene utilizzabili per l'allenamento dell'agente, insieme a diversi labirinti singoli generati
   ![image](https://github.com/user-attachments/assets/a509b19f-5738-4269-82ef-e49effb6ac17)
   ![image](https://github.com/user-attachments/assets/6878c857-31b6-49f4-91d9-e1536d62a89c)
 - Il Reward System è modificabile tramite editor. ![image](https://github.com/user-attachments/assets/61046b92-751e-4e61-97ce-a7b1f569cc67)
